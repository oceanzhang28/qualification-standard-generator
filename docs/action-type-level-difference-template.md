# 通用工作类型 × 职级差异模式

## 使用方式

本文件不再要求每个工作类型都单独填写四级差异。先沉淀两套通用职级差异模式，再让各工作类型引用对应模式。

## 工作类型与适用模式

| 工作类型 | 适用职级差异模式 |
|---|---|
| 计划类 | 模式 A：计划/规划型 |
| 执行类 | 模式 B：执行/推进型 |
| 推进类 | 模式 B：执行/推进型 |
| 协同类 | 模式 A：计划/规划型 |
| 监控类 | 模式 A：计划/规划型 |
| 复盘类 | 模式 A：计划/规划型 |
| 机制建设类 | 模式 A：计划/规划型 |
| 风险控制类 | 模式 A：计划/规划型 |
| 标准制定类 | 模式 A：计划/规划型 |

说明：
- 类别仍然保留，用于判断工作场景和句式。
- 职级差异不再逐类展开，统一引用模式 A 或模式 B。
- 如果后续发现某一类确实需要特殊规则，再单独扩展。

## 模式 A：计划/规划型

适用场景：
- 年度计划、月度计划、项目计划、资源计划、节奏规划、任务拆解。
- 协同、监控、复盘、机制建设、风险控制、标准制定等需要判断、规划、设计、优化或体系化输出的工作。

### 专员

```text
职责角色：了解该项工作是什么，参与制定，能够提出建议。
推荐动作词：了解、参与、配合、协助。
不建议动作词：主导、统筹。
行为标准示例：参与本部门专业领域人力BP，协助进行人力资源数据分析、资料收集等工作。
```

### 主管

```text
职责角色：承接专项类计划或方案制定，能够独立完成；配合重大计划制定，如部门年度计划。
推荐动作词：独立完成、主导。
不建议动作词：统筹、指导。
行为标准示例：结合业务目标，根据上年预算执行情况，独立完成月度、大促推广项目任务拆解和费用规划，支持年度计划推进。
```

### 经理

```text
职责角色：主导重大计划制定，指导专项类计划制定。
推荐动作词：组织、主导、统筹、指导。
不建议动作词：协助。
行为标准示例：主导本部门专业领域人力SP，针对本部门人力策略提供系统解决方案规划，包含市场洞察、内部洞察、识别差距和机会点、明确目标及策略、制定行动计划和资源需求规划等步骤。
```

### 总监

```text
职责角色：指导重大计划制定，深刻洞察计划影响。
推荐动作词：指导、主导、组织、领导。
不建议动作词：协助、配合。
行为标准示例：深刻理解行业人力资源内外部最佳实践和发展趋势，并据此主导外部研究，为人力规划提供研究支持。
```

## 模式 B：执行/推进型

适用场景：
- 标准动作执行、日常事务处理、资料维护、流程操作、任务交付。
- 事项推进、项目推进、节点跟踪、问题闭环、跨阶段任务落地。

### 专员

```text
职责角色：能够独立承担日常类 SOP 工作，对复杂类工作进行参与、协助。
推荐动作词：参与、协助、能够独立承担。
不建议动作词：统筹、主导。
行为标准示例：能够独立与员工进行工作对接或访谈，协助开展员工访谈调研，维护良好的员工氛围。
```

### 主管

```text
职责角色：独立完成执行性工作，是执行类工作的决定性角色。
推荐动作词：独立完成、主导推进。
不建议动作词：领导、参与。
行为标准示例：了解业务发展需求不同阶段对应的用人需求，独立开展招聘渠道分析与人才画像、人才地图建设，提出招聘渠道规划建议；承担部门岗位的招聘工作，主导招聘全流程工作。
```

### 经理

```text
职责角色：组织开展各项工作的执行，复杂工作部分独立开展。
推荐动作词：组织、主导。
不建议动作词：协助、参与。
行为标准示例：掌握部门经营指标并制定各岗位人效指标，指导或独立完成部门人效指标达成情况分析，发现人效问题并为业务提供专业建议，持续提升部门效能。
```

### 总监

```text
职责角色：推动工作问题解决，领导工作推进。
推荐动作词：领导、指导、组织、推动解决。
不建议动作词：参与、配合。
行为标准示例：指导部门干部管理与人才发展工作实施，实施高级管理干部的评估、考察工作，提出干部配置方案；引导部门进行人才盘点和人才决策，完成高潜人才的识别、跟进与培养发展。
```
