# 审核与交接门禁回归场景

以 `sparse-level-transcript.md` 的待人工审核稿为起点，分别模拟下列独立回复：

1. 含糊的通用确认：`审核通过。`
2. 明确拒绝规则补充：`确认已有岗位事实，不允许生成 Skill 对未覆盖的职级或作用范围使用内置规则补充。`

另以下列 HR 表单职级片段替换任一现有夹具的职级字段，用于模拟非标准内部职级：

```text
当前实际覆盖职级：P5、P6
受访专家主要熟悉职级：主管
```

继续分别模拟：

1. 通用确认：`审核通过。`
2. 显式映射确认：`确认将 P5 映射为专员、P6 映射为主管，适用职级范围为专员、主管。`

每个场景独立执行，不得继承其他场景的确认状态。
