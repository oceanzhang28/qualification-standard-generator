# 行为要项片段：需求转化

- 来源岗位：产品研发岗
- 行为模块：产品研发
- 学习重点：模块拆分清晰，兼具流程化和活动化；行为要项分解充分；职级差异通过动作体现；消费者洞察、基材研究、产品开发等要项颗粒度合适。

| 职级 | 行为标准 |
|---|---|
| 专员 | ① 明确需求：根据产品核心卖点需求、产品定位、产品对标竞品、产品目标成本，协助输出对标竞品测试报告。<br> ② 立项规划：参与研发立项，输出负责子项目内容规划表。 |
| 主管 | ① 明确需求：根据BP产品年度上市规划，与企划握手对齐，明确所负责产品核心卖点需求、产品定位、产品对标竞品、产品目标成本，输出所负责品项对标竞品测试报告。<br> ② 立项规划：与企划确认所负责产品需求，研发立项，输出负责产品研发规划表。 |
| 经理 | ① 明确需求：根据BP产品年度上市规划，组织小组成员与企划握手对齐，明确产品核心卖点需求、产品定位、产品对标竞品、产品目标成本，输出对标竞品测试报告。<br> ② 立项规划：与企划确认产品需求，组织小组项目成员/组为核心成员参与研发立项，输出某一品项年度产品研发项目规划书。 |
| 总监 | ① 明确需求：根据BP产品年度上市规划，领导部门成员与企划握手对齐各项目需求，领导部门成员储备新产品。<br> ② 立项规划：与企划确认产品需求，领导部门成员进行研发立项，输出卫生巾、经期裤及棉尿裤产品年度研发项目规划书。 |
