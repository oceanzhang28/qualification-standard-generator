# 行为要项片段：行业洞察

- 来源岗位：产品研发岗
- 行为模块：洞察与规划
- 学习重点：模块拆分清晰，兼具流程化和活动化；行为要项分解充分；职级差异通过动作体现；消费者洞察、基材研究、产品开发等要项颗粒度合适。
- 不建议学习：暂无

| 职级 | 行为标准 |
|---|---|
| 专员 | ① 了解卫品行业材料、技术及标准法规发展动态。 |
| 主管 | ① 行业洞察：熟悉卫品领域前沿材料、技术及标准法规发展动态，参与研讨并输出材料与技术应用可行性报告及机会点，并识别出市场、新材料、新工艺技术空白、创新机会及发展方向；<br> ② 市场洞察：参与产品的市场洞察工作，识别及洞察产品的市场机会点或升级点。 |
| 经理 | ① 行业洞察：通过参加国内外行业展会、行业协会、学术论文研讨等，深入洞察卫品领域前沿材料、技术及标准法规发展动态，作为核心成员研讨并识别出市场、新材料、新工艺技术空白、创新机会及发展方向，输出材料与技术机会点及应用可行性报告。<br> ② 市场洞察：参与产品的市场调研工作，识别及洞察产品的市场机会点或产品升级点。 |
| 总监 | ① 行业洞察：通过参加或作为演讲嘉宾参加国内外行业展会、行业协会、学术论文研讨等，深入洞察卫品领域前沿材料、技术，传播全棉改变世界理念，领导部门成员研讨并识别出市场、新材料、新工艺技术空白、创新机会及发展方向，输出材料与技术应用可行性报告及机会点，推动行业标准法规的进一步完善与优化。<br> ② 市场洞察：领导部门成员与市场研究组进行深入协同，参与产品的市场洞察工作，识别及洞察产品的市场机会点或升级点。 |
