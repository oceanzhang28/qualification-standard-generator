# 行为要项片段：产品规划

- 来源岗位：有纺产品企划岗
- 行为模块：商品企划
- 学习重点：模块拆分清晰，采用流程化方式覆盖完整工作流程。
- 不建议学习：部分职级间虽有工作细项差异，但具体工作内容存在复制情况。

| 职级 | 行为标准 |
|---|---|
| 专员 | 1、产品机会提取：根据生意规模，复盘历史品类，做出分析表格，根据市场机会洞察，提出每一季产品的1-2个机会点，并在新品种得到转化<br>2、产品结构系统维护：根据新品结构规划skc，能熟练了解主数据流程，并在PLM系统中操作 |
| 主管 | 1、各职能企划输入整合：收集和整合相关渠道对新品开发的需求<br> <br> 3、市场机会洞察：通过五看提取关键机会点，并整合<br> <br> 3、生意拆解&产品结构规划：进行三级品类、重点系列、核心爆品的生意拆解，并输出产品结构规划（skc、波段、价格等） |
| 经理 | 1、企划案框架制定：统筹组织企划组做好企划案整案的分工和研究命题，明确当季企划案的内容<br> <br> 2、各职能企划输入整合：明确渠道&职能输入信息中产品企划开发需求，拉通渠道共识新品的结构和运营策略方向<br> <br> 3、市场机会洞察：通过五看提取关键机会点，在新品企划案中体现并制定开发计划<br> <br> 3、生意拆解&产品结构规划：进行三级品类、重点系列、核心爆品的生意拆解，并输出产品结构规划（skc、波段、价格等）<br> <br> 5、核心爆品&重点系列规划：结合品类生意目标，通过数据分析、市场调研制定爆品孵化&迭代规划方案<br> <br> 6、产品开发资源需求：整合企划案达成所需资源投入，并拉通一体化职能立项达成 |
| 总监 | 1、商品企划：评估季度商品企划案符合sp&bp战略承接，针对重要产品策略做把关，确保产品结构规划、生意拆解、差异化竞争力、利润模型、品牌设计风格等对齐公司生意目标并且可持续健康发展<br> <br> 2、落地协同：有效识别产品策略执行落地中的卡点和风险点，可高效组织一体化职能正对业务痛点执定相关计划推动商品企划案的落地 |
