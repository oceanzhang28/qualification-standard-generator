# 行为要项片段：人才建设

- 来源岗位：HRBP岗
- 行为模块：业务赋能
- 学习重点：模块拆分清晰；战略规划、业务赋能、日常运作等模块贴合 HRBP 工作；职级差异明显；行为要项颗粒度合适。
- 不建议学习：无

| 职级 | 行为标准 |
|---|---|
| 专员 | 1、（人才引进）根据业务需求，承担初级岗位的招聘工作。<br>2、（干部与人才发展）参与部门干部与人才发展工作的实施，理解人才发展链路各环节的关联关系。<br>3、（春苗管理）协助进行校招生定期沟通，对接校招生实习、入职带教、定岗转正、阶段发展日常管理。 |
| 主管 | 1、（人才引进）了解业务发展需求不同阶段对应的用人需求，独立开展招聘渠道分析与人才画像、人才地图建设，提出招聘渠道规划建议；承担部门岗位的招聘工作，主导招聘全流程工作。<br>2、（干部管理）掌握新任干部三步法以及盘点IDP与继任计划的工作要求，能够独立开展部分干部管理工作实施，协助形成后备培养方案，提升内供率；<br>3、（人才发展）根据部门人才发展需求及公司制度要求，协助开展部门内人才盘点工作，提供数据等人员素材支持；能够独立开展专业力任职资格管理工作（如认证实施），进行人才梯队培养方案制定，提高部门人才密度。<br>4、（春苗管理）独立开展校招生定期沟通机制，落实校招生实习、入职带教、定岗转正、阶段发展等全过程管理，推动管培生培养目标和留存目标达成。 |
| 经理 | 1、（人才引进）理解业务发展需求不同阶段对应的用人需求，指导或独立开展招聘渠道分析与人才画像、人才地图建设，筛选有效渠道制定招聘规划并监督执行；承担核心骨干岗位的招聘工作，主导招聘全流程工作。<br>2、（干部管理）根据新任干部三步法以及盘点IDP与继任计划要求，主导部门干部管理工作实施，对部门重点岗位按标准选拔后备梯队，形成后备培养方案，提升内供率；<br>3、（人才发展）根据部门人才发展需求及公司制度要求，落地开展部门内人才盘点工作，推动业务校准盘点结果与人才决策；指导或主导专业力任职资格建设，进行人才梯队培养方案制定，提高部门人才密度。<br>4、（春苗管理）指导或主导校招生定期沟通机制，落实校招生实习、入职带教、定岗转正、阶段发展等全过程管理，推动管培生培养目标和留存目标达成。 |
| 总监 | 1、（人才引进）能够对人才猎聘和面试甄选进行前瞻性的设计，优化招聘渠道和策略；承担高级管理干部岗位的招聘工作。<br>2、（干部与人才发展）指导部门干部管理与人才发展工作实施，实施高级管理干部的评估、考察工作，提出干部配置方案，协助公司完成干部的任用；引导部门进行人才盘点和人才决策，完成高潜人才的识别、跟进与培养发展。 |
