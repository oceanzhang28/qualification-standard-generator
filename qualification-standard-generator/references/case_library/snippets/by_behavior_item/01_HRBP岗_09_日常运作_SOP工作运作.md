# 行为要项片段：SOP工作运作

- 来源岗位：HRBP岗
- 行为模块：日常运作
- 学习重点：模块拆分清晰；战略规划、业务赋能、日常运作等模块贴合 HRBP 工作；职级差异明显；行为要项颗粒度合适。
- 不建议学习：无

| 职级 | 行为标准 |
|---|---|
| 专员 | 1、（日常咨询）能够独立处理和解答员工提出的日常HR政策类问题；<br>2、（流程运作）了解HR通用专业知识，理解HR至少某2个模块专业知识。根据公司业务发展需求，在实践中运用专业知识，独立执行1-2个模块的HR SOP类工作（如绩效管理、人岗匹配、新员工管理、薪酬调整等）；<br>3、（员工关系）能够独立与员工进行工作对接/访谈，协助开展员工访谈调研，维护良好的员工氛围。 |
| 主管 | 1、（日常咨询）独立处理和解答员工提出的日常HR问题，或根据COE制定的指引解答人力资源政策实施方面的问题；<br>2、（流程运作）理解HR各模块通用专业知识，或掌握HR某模块专业知识。根据公司业务发展需求，在实践中运用专业知识，独立执行月度和年度的HR SOP类工作（如绩效管理、人岗匹配、新员工管理、薪酬调整等）；<br>3、（员工关系）与管理者和员工进行多方面的沟通，组织开展员工访谈调研，维护良好的员工氛围。 |
| 经理 | 1、（日常咨询）解答员工提出的日常HR问题，或由SSC或COE转来的复杂问题，并避免类似问题再次发生；<br>2、（流程运作）全面掌握HR各模块通用知识，或深入掌握HR某模块的专业知识。根据公司业务发展需求，公司人力资源战略规划和策略，指导他人或独立执行例行的HR SOP类工作（如绩效管理、人岗匹配、新员工管理、薪酬调整等），保证人员管理流程的有效执行；<br>3、（员工关系）与管理者和员工进行多方面的沟通，组织开展员工访谈调研，维护良好的员工氛围。 |
| 总监 | 1、（战略伙伴）出席业务部门重要的管理决策和战略制定会议，对于重要的组织、人事调整等管理活动，能够提出相关领域的专业意见并影响部门决策结果，能够获得服务的业务单元中层或以上领导团队的支持和信任<br>2、（流程运作）指导HR日常工作开展，根据需要协助处理特殊问题。 |
