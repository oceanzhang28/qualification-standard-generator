# 行为要项片段：数据管理

- 来源岗位：有纺产品企划岗
- 行为模块：产品开发
- 学习重点：模块拆分清晰，采用流程化方式覆盖完整工作流程。
- 不建议学习：部分职级间虽有工作细项差异，但具体工作内容存在复制情况。

| 职级 | 行为标准 |
|---|---|
| 专员 | 1、产品主数据管理：把控主数据整体受控进度，及时处理相关卡点的问题处理，确保产品主数据的健康运营<br> <br> 2、产品销售数据分析：了解公司数据系统，掌握相关excel数据分析工具，能够通过数据分析支撑相关产品决策 |
| 主管 | 1、产品主数据管理：把控主数据整体受控进度，及时处理相关卡点的问题处理，确保产品主数据的健康运营<br> <br> 2、产品销售数据分析：了解公司数据系统，掌握相关excel数据分析工具，能够通过数据分析支撑相关产品决策<br> <br> <br> 3、产品成本定价管理：制定产品企划成本要求，并且识别相关降本机会点 |
| 经理 | 1、主数据管理：把控主数据整体受控进度，及时处理相关卡点的问题处理，确保产品主数据的健康运营<br> <br> 2、数据分析：明确数据分析方向，提供分析思路并提出核心关注点，结合数据分析结果，制定产品升级迭代、淘汰退市、延续翻单等产品的方案<br> <br> 3、成本管理：沉淀历年的成本数据和模型，维护好企划初版成本的的合理性并识别相关风险和降本机会点 |
| 总监 | 1、分析决策：正对数据分析方案，提出相关专业性的指导意见，确认产品升级迭代、淘汰退市、延续翻单等产品的方案<br> <br> 2、成本管理：对齐公司利润指标和降本增效等战略任务，审核产品利润模型的合理性并识别相关风险和降本机会点 |
