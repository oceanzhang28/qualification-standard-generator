# 职级递进片段：市场洞察

- 来源岗位：产品设计岗
- 行为模块：产品开发
- 成熟度：不完全成熟，可对照学习
- 使用提醒：行为标准存在颗粒度、句式和职级差异不稳定的问题，不应直接作为成熟优质写法套用。

| 职级 | 行为标准 |
|---|---|
| 专员 |  |
| 主管 | 1、参与完成国内三级品类产品的市场洞察工作，输出相关市场洞察报告；识别及洞察国内某三级品类的市场机会点；<br>2、承接三级提出的市场洞察解决方案，执行设计落地 |
| 经理 | 1、承接四级市场趋势分析报告，分解和分配调研任务，组织或主导小组成员进行国内二级品类市场调研工作，参与国外市场调研工作，明确调研目的，输出国内外市场洞察计划；<br>2、制定市调计划，主导收集客户反馈，驱动产品改进：准确识别客户需求，发现改进机会；<br>承接四级提出国内外某品类的市场机会点，<br>系统且具体的提出产品解决方案，并能产生商业价值； |
| 总监 | 主导国外市场调研，从以下几个维度输出市场分析报告：<br><br>宏观环境分析：运用PEST模型（政治、经济、社会、技术）分析当前和未来的宏观环境。<br>行业分析：清晰识别行业中的生存空间、机遇和风险。需了解供应商和购买者的议价能力、新进入者的威胁、替代品的威胁以及行业内竞争的强度。<br>竞争分析：利用SWOT模型（优势、劣势、机会、威胁）分析竞争对手，识别企业的核心竞争力和潜在的市场威胁。应能够制定策略以扬长避短，抓住市场机遇。<br>市场趋势预测：能够预测市场趋势，并据此制定产品设计和创新策略。包括对消费者行为、市场细分和产品生命周期的理解。<br>战略规划与执行：需要将市场洞察转化为具体的战略规划，并能够领导团队执行这些规划。这涉及到资源分配、团队协调和项目管理的能力。<br>消费者需求分析：能够深度洞察消费者的喜好，需求和期望，敏锐捕捉需求变化趋势。指导团队制定行之有效的产品策略。 |
