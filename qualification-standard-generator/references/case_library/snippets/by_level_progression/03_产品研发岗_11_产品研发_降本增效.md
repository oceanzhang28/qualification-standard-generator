# 职级递进片段：降本增效

- 来源岗位：产品研发岗
- 行为模块：产品研发
- 成熟度：成熟
- 使用提醒：暂无

| 职级 | 行为标准 |
|---|---|
| 专员 | 根据部门降本要求及供应链需求，参与原材料或成品供应商备选/降本工作，输出降本成品性能测试数据，完成降本目标。 |
| 主管 | 根据部门降本要求及供应链需求，主导原材料或成品供应商备选/降本工作，输出降本项目计划表和降本项目关键节点，完成降本目标。 |
| 经理 | 根据BP降本要求及供应链需求，输出某一品项年度备选/降本工作计划表，管控降本项目关键节点，完成降本目标。 |
| 总监 | 根据BP降本要求及供应链需求，领导部门成员进行原材料或成品供应商备选/降本工作，输出卫生巾、经期裤及棉尿裤年度备选/降本工作计划表，领导部门成员对材料及供应商筛选和匹配确认工作，控制部门降本项目关键节点，完成降本目标。 |
