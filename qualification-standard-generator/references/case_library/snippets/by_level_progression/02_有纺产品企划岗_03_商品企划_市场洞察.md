# 职级递进片段：市场洞察

- 来源岗位：有纺产品企划岗
- 行为模块：商品企划
- 成熟度：成熟
- 使用提醒：部分职级间虽有工作细项差异，但具体工作内容存在复制情况。

| 职级 | 行为标准 |
|---|---|
| 专员 | 1、区域走访调研：制定门店大区走访计划，记录相关走访的调研内容并整合输出报告<br> 2、线上竞品调研：使用情报通、知衣、生意参谋等工具，对线上竞品进行调研分析<br> 具体包括：1.竞品调研；2、行业&科技调研3、消费者（消费者趋势出生率、消费趋势、购物习惯、频次&本品会员分析、机会）4、本品优势和差距、机会 |
| 主管 | 1、区域走访调研：制定门店大区走访计划，记录相关走访的调研内容并整合输出报告<br> 2、线上竞品调研：使用情报通、知衣、生意参谋等工具，对线上竞品进行调研分析<br> 具体包括：1.竞品调研；2、行业&科技调研3、消费者（消费者趋势出生率、消费趋势、购物习惯、频次&本品会员分析、机会）4、本品优势和差距、机会 |
| 经理 | 1、区域走访&线上竞品调研：运用五看工具，输出调研报告，并整合在新一季开发中<br> <br> 2、行业趋势洞察：结合行业权威趋势报告&内部消费者调研小组、战规的市场报告，上市财报，洞察行业未来增长机会和产品开发机会点，转化进新品开发中<br> <br> 3、消费者调研：联合调研专家、小组、机构针对产品问题、消费者画像等调研需求，制定相关调研方案，确认调研方向、问卷设计、座谈会等调研形式来达到产品 |
| 总监 | 1、行业洞察：能从行业趋势专家、产品专家、标杆团队等专业权威为团队输入行业报告并识别机会点，同时可以组织相关学习交流的专业研讨会<br> <br> 2、机会识别：具备前瞻性可通过五看预测未来1-2年的产品趋势，提前布局趋势赛道的产品，同时有敏锐的洞察力和执行力可以抓住当前的机会并组织一体化职能协同实现商机变现 |
