# 职级递进片段：基材研究

- 来源岗位：产品研发岗
- 行为模块：基础研究
- 成熟度：成熟
- 使用提醒：暂无

| 职级 | 行为标准 |
|---|---|
| 专员 | ① 基材研究：通过精通掌握表面化学、有机化学、物理化学、非织造学等专业知识，深入研究卫生用品13款基础材料（吸收体+非吸收体+闭合体）组成成分、生产工艺以及常规检测项目原理。<br> ② 项目管理：对负责的基础材料研究项目进行项目管理，输出项目进度表。 |
| 主管 | ① 基材研究：针对某一创新基材的研究需求（SP/BP），通过精通掌握表面化学、有机化学、物理化学、非织造学等专业知识，以及对基材研究背后的科学原理解析，协助研究核心差异化纯棉基材，独立研究基础材料，并成功应用在产品上实现产品创新差异化以及一定的技术壁垒。<br> ② 项目管理：作为项目核心成员主导/参与某一子项目并进行项目管理，定期复盘项目进度及技术瓶颈，研讨切实有效的解决方案并熟练运用PDCA等工具跟踪控制，输出项目进度表。<br> ③ 风险管控：作为项目核心成员参与复盘项目质量风险、成本风险以及进度风险，输出项目风险评估表，并参与研讨风险应对方案。 |
| 经理 | ① 基材研究：针对某一创新基材的研究需求（SP/BP），通过精通掌握表面化学、有机化学、物理化学等专业知识，以及对基材研究背后的科学原理解析，来独立立项攻关行业颠覆式/核心差异化纯棉基材，并成功应用在产品上实现产品创新差异化以及一定的技术壁垒。<br> ② 项目管理：定期组织部门/小组成员/作为项目核心成员主导复盘项目进度及技术瓶颈，研讨切实有效的解决方案并熟练运用PDCA等工具跟踪控制，输出项目进度表。<br> ③ 风险管控：定期组织部门/小组成员/作为项目核心成员主导复盘项目质量风险、成本风险以及进度风险，输出风险应对方案。 |
| 总监 | ① 基材研究：针对SP/BP规划创新基材的研究需求，依据吸收体吸收模型、以及基础材料测评体系，通过精通掌握表面化学、有机化学、物理化学等专业知识，以及对基材研究背后的科学原理解析，领导部门成员深入研究创新棉质基材，并成功应用在产品上实现产品创新差异化以及一定的技术壁垒。<br> ② 项目管理：领导部门成员复盘项目进度及技术瓶颈，研讨切实有效的解决方案并熟练运用PDCA等工具跟踪控制，输出项目进度表。<br> ③ 风险管控：领导部门成员复盘项目质量风险、成本风险以及进度风险，评估确认风险应对方案。 |
