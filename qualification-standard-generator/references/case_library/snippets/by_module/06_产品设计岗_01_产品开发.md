# 行为模块片段：产品开发

- 来源岗位：产品设计岗
- 成熟度：不完全成熟，可对照学习
- 学习重点：可参考其产品开发、产品管理、项目管理、组织贡献的精简模块结构，以及产品设计类工作细项拆分。
- 不建议学习：行为标准存在颗粒度、句式和职级差异不稳定的问题，不应直接作为成熟优质写法套用。

| 原始序号 | 行为要项 | 专员行为标准 | 主管行为标准 | 经理行为标准 | 总监行为标准 |
|---|---|---|---|---|---|
| 1 | 产品开发策略 |  | / | 参与制定基于产品层面的开发策略，聚焦单/多品类产品的策略性规划，通过市场调研与产品分析，提炼差异化机会点，拆解策略为可执行节点落地。 | 通过趋势预判、组织协同，聚焦行业品牌及全品类产品矩阵的战略性洞察，研究行业宏观趋势，每季通过制定相应的产品开发策略，逐步构建品牌产品长期路线图，提升品牌的长期产品竞争力。 |
| 2 | 市场洞察 |  | 1、参与完成国内三级品类产品的市场洞察工作，输出相关市场洞察报告；识别及洞察国内某三级品类的市场机会点；<br>2、承接三级提出的市场洞察解决方案，执行设计落地 | 1、承接四级市场趋势分析报告，分解和分配调研任务，组织或主导小组成员进行国内二级品类市场调研工作，参与国外市场调研工作，明确调研目的，输出国内外市场洞察计划；<br>2、制定市调计划，主导收集客户反馈，驱动产品改进：准确识别客户需求，发现改进机会；<br>承接四级提出国内外某品类的市场机会点，<br>系统且具体的提出产品解决方案，并能产生商业价值； | 主导国外市场调研，从以下几个维度输出市场分析报告：<br><br>宏观环境分析：运用PEST模型（政治、经济、社会、技术）分析当前和未来的宏观环境。<br>行业分析：清晰识别行业中的生存空间、机遇和风险。需了解供应商和购买者的议价能力、新进入者的威胁、替代品的威胁以及行业内竞争的强度。<br>竞争分析：利用SWOT模型（优势、劣势、机会、威胁）分析竞争对手，识别企业的核心竞争力和潜在的市场威胁。应能够制定策略以扬长避短，抓住市场机遇。<br>市场趋势预测：能够预测市场趋势，并据此制定产品设计和创新策略。包括对消费者行为、市场细分和产品生命周期的理解。<br>战略规划与执行：需要将市场洞察转化为具体的战略规划，并能够领导团队执行这些规划。这涉及到资源分配、团队协调和项目管理的能力。<br>消费者需求分析：能够深度洞察消费者的喜好，需求和期望，敏锐捕捉需求变化趋势。指导团队制定行之有效的产品策略。 |
| 3 | 设计企划 |  | 3、承接三级品类中的某系列设计企划案/设计风格/颜色/面料等内容的输出 | 3、根据商品企划及设计企划，制定三级各品类设计企划主题方向/设计风格/颜色/面料等大框架，并向四级汇报。 | 战略规划：<br>将公司战略目标转化为具体设计目标，能够制定出符合公司目标的设计企划。确保设计团队的工作与公司的整体战略保持一致，同时推动创新和差异化。<br>创意领导力：<br>有强大的创意领导力，能够激发团队的创造力和创新思维。<br>引导团队探索新的设计理念，制定整个商品中心设计大主题框架，同时确保这些理念能够转化为实际的产品设计。<br>沟通与协调能力：<br>能够与团队成员、其他部门以及外部合作伙伴进行有效沟通。清晰地传达设计理念和目标，确保所有相关方都对项目有清晰的理解。协调不同团队和资源，以确保设计项目的顺利进行。<br>用户洞察与市场理解：<br>对用户需求和市场动态有深刻的理解，通过用户研究和市场分析，识别产品的目标用户群体和市场需求。将这些洞察转化为设计决策，确保产品设计能够满足用户需求和市场趋势。<br>市场导向：<br>基于市场洞察指导设计企划，了解消费者需求，竞争对手动态，使设计企划能够贴合市场实际需求且能有效预估设计企划在市场中反应，提前做好风险评估和应对策略<br>数据分析与运用：<br>具有收集和分析设计相关数据能力，利用数据优化设计企划方案，通过数据评估设计企划有效性。 |
| 4 | 产品设计 |  | 4、承接三级品类某系列开发工作（面辅料开发/颜色花型设计/款式设计），完成需求分解，符合成本目标管控<br><br><br>5、针对自己负责的单品设定迭代目标，执行迭代落地方案，组织市场调研，协助评估迭代效果<br><br>6、产品开发资料下发规范无误， 及时向上反馈影响开发的风险点（质量、成本、进度）并执行三级提出的解决方案。 | 4、主导组内成员的设计分工（面辅料开发/颜色花型设计/系列款式设计），<br>     承接某三级品类开发工作，符合成本目标管控及整体货盘规划。<br><br>5、针对小组重点产品、重点链接，设定长期和短期的迭代目标；提出迭代决策，制定短期迭代落地方案；在迭代结束后，需制定用户、市场调研计划评估迭代效果<br><br>6、监督产品开发过程中的实施阶段，识别影响产品开发的风险点，输出解决方案<br><br>7、有创新研究开发能力以及商业转化的产品开发能力，并有实际落地的爆品开发方案和创新成功案例（创新突破力） | 1：根据已确定的设计企划，输出面辅料前沿使用趋势，制定季度面辅料开发方向、审核面辅料落地方案，符合成本目标管控及整体货盘规划。<br>2：监督设计落地过程、把控款式、颜色、花型与品类系列、整体产品契合度, 组织部门设计内审会，输出明确调整意见<br>3：拉通不同品类、部门对于设计方向的一致性，并跨部门组织设计拉通会议，并输出整体调整意见<br><br>4:监督部门产品开发过程中的实施阶段，预判影响部门内开发的各风险点（质量、成本、进度），组织三级梳理组内、系列内产品风险点，并组织跨部门联动会议化解风险点（质量、采购、研发）。<br>5、有敏锐嗅觉，市场感知力，发现大单品、单爆品的机会，并输出灵感赋能，组织部门相关核心人员的讨论，制定进一步施行策略 |
| 5 | 产品评审会、订货会 |  | 7、参与评审会订货会工作，并执行上级提出的产品改进方案，保证个人负责样品与设计稿的统一性，完成本产品组的各季原样和齐色评审会筹备、产品设计发布、复版，输出设计定稿、封样版等相关工作。<br><br>8、完成产品卖点培训资料输出：设计信息传达准确，层次清晰，方案符合标准。<br><br>9、对产品拍摄搭配方案提出修改意见并现场跟进拍摄。 | 8、主导评审会订货会，制定工作计划，保证顺利开展，根据各渠道评审意见，从本专业的角度提出改进意见和解决方案。<br><br>9、制定卖点资料标准，组织或指导产品卖点培训资料输出，持续收集市场反馈，并根据上级给出的策略性调整意见进行产品卖点的迭代升级<br><br>10、整合产品拍摄搭配方案的修改意见，跟拍分工，各导视实物图色彩成果验收。 | 6、统筹部门产品两评两审，根据渠道意见，确定主推品、各套餐，协同市场营销资源，确定展示和陈列形式。<br><br>7、根据卖点信息的传播效果，市场反馈，给出策略性调整性意见，提升产品价值传递的有效性以及产品商业价值与市场吸引力，并对信息传播的利益风险点、舆论风险进行评估预判<br><br>8、输出本季产品拍摄风格方向，确定产品端拍摄调性的需求，并组织协同内容、营销、视觉讨论整体拍摄方案，并监督拍摄方案的执行 |
