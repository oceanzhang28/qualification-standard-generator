# 通用工作类型职级差异模式

本文件用于生成行为标准时选择职级动作和职责差异。工作类型可以继续细分，但职级差异优先引用以下两套通用模式。

## 工作类型与模式映射

| 工作类型 | 适用模式 |
|---|---|
| 计划类 | 模式 A：计划/规划型 |
| 执行类 | 模式 B：执行/推进型 |
| 推进类 | 模式 B：执行/推进型 |
| 协同类 | 模式 A：计划/规划型 |
| 监控类 | 模式 A：计划/规划型 |
| 复盘类 | 模式 A：计划/规划型 |
| 机制建设类 | 模式 A：计划/规划型 |
| 风险控制类 | 模式 A：计划/规划型 |
| 标准制定类 | 模式 A：计划/规划型 |

## 模式 A：计划/规划型

适用：计划、规划、协同、监控、复盘、机制建设、风险控制、标准制定等需要判断、设计、优化或体系化输出的工作。

| 职级 | 职责角色 | 推荐动作词 | 不建议动作词 |
|---|---|---|---|
| 专员 | 了解该项工作是什么，参与制定，能够提出建议 | 了解、参与、配合、协助 | 主导、统筹 |
| 主管 | 承接专项类计划或方案制定，能够独立完成；配合重大计划制定 | 独立完成、主导 | 统筹、指导 |
| 经理 | 主导重大计划制定，指导专项类计划制定 | 组织、主导、统筹、指导 | 协助 |
| 总监 | 指导重大计划制定，深刻洞察计划影响 | 指导、主导、组织、领导 | 协助、配合 |

示例：
- 专员：参与本部门专业领域人力 BP，协助进行人力资源数据分析、资料收集等工作。
- 主管：结合业务目标，根据上年预算执行情况，独立完成月度、大促推广项目任务拆解和费用规划，支持年度计划推进。
- 经理：主导本部门专业领域人力 SP，针对本部门人力策略提供系统解决方案规划。
- 总监：深刻理解行业人力资源内外部最佳实践和发展趋势，并据此主导外部研究，为人力规划提供研究支持。

## 模式 B：执行/推进型

适用：标准动作执行、日常事务处理、资料维护、流程操作、任务交付、事项推进、节点跟踪、问题闭环。

| 职级 | 职责角色 | 推荐动作词 | 不建议动作词 |
|---|---|---|---|
| 专员 | 能够独立承担日常类 SOP 工作，对复杂类工作进行参与、协助 | 参与、协助、能够独立承担 | 统筹、主导 |
| 主管 | 独立完成执行性工作，是执行类工作的决定性角色 | 独立完成、主导推进 | 领导、参与 |
| 经理 | 组织开展各项工作的执行，复杂工作部分独立开展 | 组织、主导 | 协助、参与 |
| 总监 | 推动工作问题解决，领导工作推进 | 领导、指导、组织、推动解决 | 参与、配合 |

示例：
- 专员：能够独立与员工进行工作对接或访谈，协助开展员工访谈调研，维护良好的员工氛围。
- 主管：了解业务发展需求不同阶段对应的用人需求，独立开展招聘渠道分析与人才画像、人才地图建设，提出招聘渠道规划建议；承担部门岗位的招聘工作，主导招聘全流程工作。
- 经理：掌握部门经营指标并制定各岗位人效指标，指导或独立完成部门人效指标达成情况分析，发现人效问题并为业务提供专业建议，持续提升部门效能。
- 总监：指导部门干部管理与人才发展工作实施，实施高级管理干部的评估、考察工作，提出干部配置方案；引导部门进行人才盘点和人才决策，完成高潜人才的识别、跟进与培养发展。
