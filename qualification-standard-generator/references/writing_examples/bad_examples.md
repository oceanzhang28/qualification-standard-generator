# 常见错误写法

## 空泛能力

错误：

```text
具备良好的沟通能力。
有较强的数据分析能力。
熟悉业务流程。
```

修订方向：

```text
写清楚基于什么信息，完成什么动作，形成什么产出，如何判断达标。
```

## 态度和价值观

错误：

```text
工作积极主动。
责任心强。
能够吃苦耐劳。
```

修订方向：

```text
改成具体工作行为和交付要求，不评价态度。
```

## 职责罗列

错误：

```text
负责产品推广。
负责数据分析。
负责项目推进。
```

修订方向：

```text
说明工作输入、关键动作、产出和评价标准。
```

## 职级无差异

错误：

```text
专员：完成数据分析。
主管：独立完成数据分析。
经理：统筹完成数据分析。
总监：指导完成数据分析。
```

问题：

```text
只有动词变化，没有说明责任范围、复杂度、产出形态和业务影响差异。
```

修订方向：

```text
按执行、模块负责、跨模块统筹、体系建设和经营支撑拉开差异。
```

## 把协作方职责写成本岗位职责

错误：

```text
推动设计团队完成视觉方案。
```

修订方向：

```text
如果本岗位不是设计岗位，应写成本岗位的需求澄清、资料输出、评审反馈或协同推进动作。
```
