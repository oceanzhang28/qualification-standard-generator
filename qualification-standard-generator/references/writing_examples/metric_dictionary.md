# 评价词典

行为标准必须可评价，但不强制量化。本文件提供常用评价表达。

## 量化评价

| 类型 | 可用表达 |
|---|---|
| 时效 | 按时完成、按节点交付、响应及时 |
| 准确 | 数据准确、口径一致、差错可控 |
| 完整 | 信息完整、材料齐备、覆盖关键场景 |
| 达成 | 目标达成、计划完成、结果符合要求 |
| 合规 | 符合流程、符合制度、符合标准 |
| 覆盖 | 覆盖核心人群、覆盖关键渠道、覆盖重点场景 |

## 定性评价

| 类型 | 可用表达 |
|---|---|
| 判断质量 | 判断清晰、问题识别准确、原因分析合理 |
| 方案质量 | 方案可落地、措施具体、资源需求清晰 |
| 决策支撑 | 能支撑业务决策、能支撑后续动作、能形成管理依据 |
| 过程闭环 | 问题闭环、责任清晰、过程可追踪 |
| 风险控制 | 风险识别充分、预警及时、处置措施有效 |
| 机制沉淀 | 可复制、可推广、可持续迭代 |

## 使用提醒

- 不要为了量化而强行编造指标。
- 如果工作本身难以量化，使用清晰的定性评价。
- 定性评价也必须能被 HR 和业务复核。
