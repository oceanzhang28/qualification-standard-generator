# 职级差异规则

## 固定职级

本 Skill 固定使用四个职级：

```text
专员
主管
经理
总监
```

最终表格始终保留四个职级列。若用户选择的适用职级不是完整四级，未选择的职级列留空。

## 专员

定位：

```text
初做者
```

整体描述：

```text
具备基础的专业知识，在工作中多次得以实践，在例行情况下能基于岗位职责和任务要求，独立完成常规工作，在关键性的指导下能完成较复杂工作。
```

关键词：

```text
了解、参与、配合、协助、提出建议、执行、收集、整理、跟进、维护、初步分析、基础判断、按标准完成
```

典型定位：

```text
在明确规则和上级指导下，完成具体工作项的执行、资料整理、数据维护、基础分析和过程跟进，确保结果准确、及时、符合流程要求。
```

写作要点：
- 描述具体执行动作。
- 可以体现基础判断，但不应承担复杂方案制定或统筹职责。
- 应强调准确、及时、按流程、按标准。
- 解决问题：通过请教或资料查询可直接解决常规问题。
- 知识深度：了解。
- 专业影响力：被指导。

## 主管

定位：

```text
独立任务落地
```

整体描述：

```text
具备扎实的专业知识，能独立、高效地完成所有工作内容，独立分析并完成较复杂工作，并能有效对初做者进行一定程度的技术或专业上的指导。
```

关键词：

```text
理解、参与、配合、独立完成、独立负责、协助、提供解决方案、分析判断、计划制定、过程推动、问题处理、跨部门协同
```

典型定位：

```text
能够独立负责某一模块或专项工作，基于业务目标进行分析判断、计划制定和过程推进，协调相关方解决常规问题，确保模块目标按要求达成。
```

写作要点：
- 描述独立负责某模块或专项。
- 可以制定计划、推动过程、处理常规问题。
- 可体现与相关方协同，但不应写成泛化沟通能力。
- 解决问题：独立分析并解决问题。
- 知识深度：掌握。
- 专业影响力：小组专业指导。

## 经理

定位：

```text
骨干
```

整体描述：

```text
精通本专业知识，能够跟踪本行业的发展动态，解决本领域常规的复杂问题，进行本领域体系或机制的建立或优化，体系化提供本专业领域解决方案。
```

关键词：

```text
掌握、主导、统筹、配合、独立开展、承接、指导、规划、资源配置、机制优化、复杂问题解决、结果负责
```

典型定位：

```text
能够统筹多个模块或复杂专项，围绕业务目标制定策略与实施方案，协调资源、推动跨部门协同并解决关键问题，对工作结果和业务影响负责。
```

写作要点：
- 描述跨模块统筹、复杂问题解决和机制优化。
- 强调对结果负责。
- 不写成单纯人员管理或领导力标准。
- 解决问题：拥有完整解决方案能力，推动解决复杂问题。
- 知识深度：精通。
- 专业影响力：部门专业指导。
- 不得所有行为标准都以“统筹”开头，应根据工作类型选择主导、制定、判断、优化、解决、推动、把关、沉淀等动作。

## 总监

定位：

```text
体系/内部专家
```

整体描述：

```text
对本专业及相关专业知识能够融会贯通，能够有效洞察本行业的发展动态，分析并推动解决非常规的、复杂的问题；引领业务、技术和专业解决方案创新及体系搭建，为公司本领域的专业或技术产生关键影响。
```

关键词：

```text
深刻洞察、掌握、主导、指导、策略制定、体系建设、方向判断、组织协同、重大决策、经营结果
```

典型定位：

```text
能够从业务战略和组织能力建设角度，制定专业领域方向、规则和机制，推动跨组织协同与资源配置，对专业体系成熟度和经营结果产生关键影响。
```

写作要点：
- 描述专业方向、体系建设、重大决策支持、经营结果影响。
- 可体现组织协同，但必须落在专业工作项和专业机制上。
- 不写成通用领导力、团队管理意识或价值观要求。
- 解决问题：创新性推动解决非常规复杂问题。
- 知识深度：融会贯通。
- 专业影响力：公司专业指导。

## 动作词使用原则

- 各职级动作词只是常用项，不是固化模板。
- 总监、经理也会有配合事项，例如配合公司整体或其他部门工作，需要根据具体工作在该层级的真实角色选择动作。
- 同一职级在不同工作项下应体现不同责任和动作，不得高频重复同一动作词。
- 行为动词要与职级定位、工作内容难度、责任范围和产出要求一致。

## 职级递进检查

四级之间必须体现至少一种递进：

- 责任范围扩大。
- 问题复杂度提升。
- 影响范围扩大。
- 决策权限提升。
- 资源协调范围扩大。
- 从执行到独立负责、统筹规划、体系建设。

如果四级只是替换动词、没有责任和复杂度差异，必须修订。
