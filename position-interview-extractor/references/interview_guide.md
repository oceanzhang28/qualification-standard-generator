# 岗位关键任务访谈提纲

## 访谈目标与时长

用约 90 分钟了解单个专业类岗位的实际工作、关键任务和专家能够确认的职级差异。访谈聚焦岗位事实，不讨论学历、工作年限、性格、通用能力、领导力、价值观或人员管理要求。

| 时间 | 环节 | 目标 |
| --- | --- | --- |
| 0–5 分钟 | 开场与口径 | 说明访谈范围和确认方式 |
| 5–15 分钟 | 岗位定位 | 识别价值、对象、输入、产出与边界 |
| 15–30 分钟 | 工作全景 | 形成 5–10 项候选关键任务 |
| 30–65 分钟 | 逐项深挖 | 获取每项任务的事实证据 |
| 65–80 分钟 | 职级差异 | 记录专家熟悉范围内的真实差异 |
| 80–87 分钟 | 组织贡献与遗漏 | 识别机制建设、人员培养和遗漏事项 |
| 87–90 分钟 | 复述确认 | 校正任务、边界、常态性和职级信息 |

## 开场话术

> 今天主要了解这个岗位实际承担的专业工作。请尽量结合真实场景说明：基于什么、做什么、产出什么、怎样算做好。后面会询问不同职级的工作差异；只需回答您熟悉的部分，不确定的职级可以直接说明。访谈内容将先被整理为岗位事实审核稿，供 HR 或岗位专家确认。

## 岗位定位（5–15 分钟）

1. 这个岗位存在的核心价值是什么，主要解决哪些业务问题？
2. 主要服务或作用于哪些对象？
3. 通常接收哪些上游输入或依据？
4. 最终交付或沉淀哪些成果？
5. 哪些工作容易与其他岗位混淆？本岗位主责什么、不负责什么？

## 工作全景（15–30 分钟）

1. 按一个完整业务周期看，工作通常经过哪些步骤？
2. 最重要的 5–10 项工作是什么？哪些不做就无法体现岗位核心价值？
3. 哪些是周期性工作，哪些由项目、问题或突发事件触发？
4. 哪些任务最影响业务结果，或最需要专业判断？

可按年度、季度、月度、周度，事前规划—过程推进—结果复盘，最近典型项目或主要产出物帮助回忆。访谈者当场复述候选任务清单，先确认全景，不急于改写为任职资格语言。

## 逐项深挖关键任务（30–65 分钟）

从候选任务中选择约 5–8 项，每项约 4–6 分钟；按已获得信息选择追问，无须机械问完全部问题。

| 维度 | 追问 |
| --- | --- |
| 任务与场景 | 这项任务具体做什么、通常如何称呼？在什么情况下启动，是固定周期、业务需求、项目节点还是问题触发？ |
| 输入与依据 | 开始前需要哪些目标、需求、数据、规则、研究或其他输入？哪些最影响判断？ |
| 关键动作 | 从接到任务到完成的关键步骤是什么？哪些步骤最需专业判断，哪些决定质量？ |
| 产出与评价 | 最终形成什么成果或完成状态？怎样算做好？常见质量问题、风险或返工原因是什么？ |
| 协同与边界 | 本岗位主责什么？协作方分别提供、决定和交付什么？分歧或异常时谁判断、推动并对结果负责？ |
| 复杂度 | 常规情况如何处理？复杂情形是什么？复杂时动作、判断、协同范围或产出要求如何变化？ |

必要时请专家讲述最近实例，用于校验任务描述是否具体；偶发案例不等同于岗位常态。

## 稀疏职级差异追问（65–80 分钟）

只记录专家能够确认的职级信息，不要求补满专员、主管、经理、总监四个职级。

1. 您熟悉哪些职级？哪些无法准确判断？
2. 哪一级能够独立负责完整任务？哪些职级负责复杂专项、跨模块统筹、机制优化、专业方向或体系建设？
3. 对差异明显的任务，实际差异体现在哪些方面：任务范围、问题难度、判断权限、影响范围、产出层级或协同范围？

先请专家描述真实差异，再用上述维度查漏；不要诱导其套用固定职级词。无法确认的职级应在转写中明确标记为缺失，不能因未覆盖而认定不适用。

## 组织贡献与遗漏（80–87 分钟）

1. 该岗位会参与或负责哪些工具、模板、流程、方法、知识库或专业机制建设？
2. 该岗位如何进行经验分享、专业指导、案例沉淀或新人培养？
3. 上述工作在哪些职级真实存在？不清楚的部分请明确说明。
4. 是否遗漏低频但影响重大的任务、风险处理、复盘优化或专业判断类任务？是否误把协作方职责算为本岗位主责？哪些只是临时事项？

## 复述确认（87–90 分钟）

复述核心任务、工作对象、主要流程、产出物、主责边界，以及已确认和未确认的职级信息。请专家校正任务名称、合并或拆分、主责与协同边界、个别案例与岗位常态的区别。

## 转写主题标记

访谈者在主题切换时尽量使用完整句：

- “下面讨论关键任务一：……”
- “这项任务的主要产出是……”
- “这里补充本岗位与协作方的边界……”
- “下面比较这项任务的职级差异……”
- “这一职级专家无法确认……”
- “以上内容属于个别案例，不代表岗位常态……”

这些标记帮助从完整转写稿中划分主题，不要求另做逐项表格记录。
