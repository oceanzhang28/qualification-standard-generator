# HR 岗位基本信息表

```text
【岗位基本信息】
职类：
职务子类：
所在部门：
岗位名称：
岗位性质（仅可选：专业类岗位 / 管理岗位）：
当前实际覆盖职级（仅可多选：专员、主管、经理、总监）：
待映射的内部职级（如 P5、M2；无则填“不需要”）：
受访专家姓名/角色：
受访专家主要熟悉职级：
访谈日期：
```

“岗位性质”是阻断字段，只接受“专业类岗位”或“管理岗位”。本 Skill 仅处理已明确确认为“专业类岗位”的输入；不得根据岗位名称、职类或访谈内容推断岗位性质。

“当前实际覆盖职级”描述岗位在组织中的实际设置，交接时只接受固定四级。如果组织使用其他职级口径，将原值填入“待映射的内部职级”，后续由审核人明确确认映射。“受访专家主要熟悉职级”只描述证据覆盖范围，不得用于映射岗位职级；专家不熟悉某职级，不表示该职级未设置或不适用。
